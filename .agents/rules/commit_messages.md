# Langue des messages de commit

## Règle
Tous les messages de commit doivent impérativement être rédigés en français, tout en respectant la spécification Conventional Commits.

## Exemples
- `feat: configuration du pool PostgreSQL`
- `feat: typage des modèles de données`
- `fix: correction de la connexion au pool`
- `chore: validation de l'environnement`
- `docs: mise à jour du fichier de suivi`
